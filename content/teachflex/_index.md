Teachflex is a modern, responsive HTML template designed for online learning platforms, bootcamps, training centers, and educators. It is the HTML version of the [**Teachflex â€“ Online Education & Bootcamp Elementor Template Kit**](https://themeforest.net/item/teachflex-online-education-bootcamp-elementor-template-kit/59427478) on ThemeForest, adapted for a pure HTML/CSS/JS workflow.

This documentation covers everything you need to install, customize, and deploy the template, and it is also included as an offline HTML documentation inside your download package.

## Folder structure overview

After extracting the **Main Files** download from ThemeForest, you will see a structure similar to:

- `html/` - Main HTML template folder.
- `documentation/` - Offline HTML documentation that you can open in your browser (no internet connection required).

Inside the `teachflex` folder, the core structure is:

- HTML pages:

  - `index.html` - Main homepage.
  - `about-us.html` - About page.
  - `bootcamp.html`, `bootcamp-details.html` - Bootcamp listing and single page.
  - `blog-list.html`, `blog-details.html` - Blog listing and article page.
  - `pricing.html` - Pricing and plans page.
  - `testimonials.html` - Testimonials page.
  - `faq.html` - FAQ page.
  - `contact.html` - Contact page.
  - `404-page.html` - 404 not found page.

- Assets:
  - `assets/css/` - All CSS files, including:
    - `bootstrap.min.css`
    - `bootstrap-icons.min.css`
    - `swiper-bundle.min.css`
    - `aos.css`
    - `main.css` (main custom stylesheet)
  - `assets/js/` - All JavaScript files:
    - `bootstrap.bundle.min.js`
    - `swiper-bundle.min.js`
    - `aos.js`
    - `jquery-3.7.1.min.js`
    - `main.js` (main custom script)
  - `assets/fonts/` - Local font files for Bootstrap Icons.
  - `assets/img/` - All images and graphics used in the template (logos, illustrations, course images, avatars, etc.).

You can upload the `teachflex` folder directly to your hosting, or place its contents inside your existing project structure.

## Tech stack

Teachflex HTML is built with:

- Bootstrap v5.3.8 for layout and responsive grid.
- Bootstrap Icons v1.13.1 for iconography.
- Swiper JS v12.0.3 for sliders and carousels.
- AOS (Animate On Scroll) for scroll-based animations.

No build tools (Vite, SCSS, etc.) are required to use this template in production; you only work with the provided HTML, CSS, and JS files.

## Offline documentation

For your convenience, an offline HTML version of this documentation is included in the `documentation` folder.
